Hacky Hour

 2015-ongoing

A meetup/helpdesk for researchers starting in the data science field. Description:
This initiative was set up to compliment the release of 'Intro to Programming for Researchers' classes. During my work in Bioinformatics, I realised there was no real support for researchers to learn how to deal with large amounts of data, or needing to learn data science. This was reflected in groups of researchers reaching out and sharing the same experiences. I established numerous workshops to meet these needs, and started the "Hacky Hours" to support researchers outside of these workshops. We became the place researchers came to ask about machine learning, high performance compute, large data storage, what programming language to start with and help with debugging. This initiative was modelled on UniMelb's Hacky Hour, but altered to suit our local needs.
